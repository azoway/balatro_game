# 小丑牌 · JOKER

Balatro 风格的纯前端网页卡牌游戏，无任何依赖，打开 `index.html` 即玩。

**在线试玩:** https://azoway.github.io/balatro_game/

## 玩法

- 从 8 张手牌中选至多 5 张出牌，按扑克牌型计分（筹码 × 倍率）
- 每个底注需依次击败小盲注 / 大盲注 / Boss 盲注（Boss 带随机减益），共 8 个底注
- 回合胜利后进入商店：购买小丑牌（被动加成）和星球牌（升级牌型）
- 金钱有利息（每 $5 得 $1，上限 $5）

## 操作

| 按键 | 功能 |
|------|------|
| 鼠标点击 / `1`-`9` | 选牌 |
| `Enter` | 出牌 |
| `X` | 弃牌 |
| `R` / `S` | 按点数 / 花色排序 |
| `M` | 静音 |
| 计分中点击屏幕 | 4 倍速快进 |

进度自动保存在浏览器 localStorage，刷新后可从盲注选择处继续。
