# 小丑牌 · JOKER

Balatro 风格的纯前端网页卡牌游戏，无任何依赖，打开 `index.html` 即玩。中/英双语（侧边栏切换）。支持 PWA：部署到 HTTPS 后可安装到桌面/手机并离线游玩。

**在线试玩:** https://azoway.github.io/balatro_game/

## 玩法

- 从 8 张手牌中选至多 5 张出牌，按扑克牌型计分（筹码 × 倍率）
- 每个底注需依次击败小盲注 / 大盲注 / Boss 盲注（Boss 带随机减益且同局不重复），共 8 个底注；通关后可进入无尽模式，目标分数持续增长
- 回合胜利后进入商店：小丑牌（被动加成，可用 ◀▶ 调整结算顺序）、星球牌（升级牌型）、塔罗牌（消耗品）、卡包（3 选 1）、优惠券（永久升级：槽位 / 刷新费 / 利息上限等，每种每局限购一次）
- 塔罗牌分两类：即时型（金钱翻倍、出牌/弃牌/手牌上限 +1 等）和目标型（回合中选中手牌后使用：改花色、点数 +1、附加增强、销毁牌）
- 卡牌增强 4 种：加成牌（+30 筹码）、倍率牌（+4 倍率）、钢铁牌（留在手中 ×1.5 倍率）、黄金牌（回合结束在手中 +$3）
- 牌库整局持久，点击右下角「牌库」可随时查看整副牌的现状与增强
- 跳过小盲注 / 大盲注可获得随机标签奖励（金钱 / 免费刷新 / 牌型升级）
- 金钱有利息（每 $5 得 $1，上限 $5，可用优惠券提到 $10）

## 操作

| 按键 | 功能 |
|------|------|
| 鼠标点击 / `1`-`9` | 选牌 |
| `Enter` | 出牌 |
| `X` | 弃牌 |
| `R` / `S` | 按点数 / 花色排序 |
| `M` | 静音（自动记忆） |
| 计分中点击屏幕 | 4 倍速快进 |
| 点击消耗品 ×2 | 使用塔罗牌 |
| 点击小丑牌 ×2 | 卖出；悬停出现 ◀▶ 可调顺序 |

进度自动保存在浏览器 localStorage（含回合中的手牌与牌库），刷新后可从中断处继续。跨局战绩与小丑图鉴见侧边栏「图鉴」。

## 种子与每日挑战

牌局随机使用可播种 RNG（mulberry32），同一种子可复现整局。侧边栏「今日挑战」按当天日期生成固定种子；点击「种子」一行可输入自定义种子开局。

## 结构

```
index.html    页面骨架
style.css     样式
i18n.js       国际化（zh/en 字典、L/S 工具）
defs.js       数据定义（牌型 / Boss / 小丑 / 星球 / 塔罗 / 卡包 / 优惠券 / 标签）
engine.js     引擎（RNG / 状态 / 存档 / 牌型判定 / 纯函数计分 / 卡包 / 跨局统计）
ui.js         渲染 / 动画 / 音效 / 流程 / 事件绑定
sw.js         Service Worker（网络优先、缓存兜底）
manifest.json PWA 清单
```
